# Brafe

Projeto desenvolvido durante o curso de **CSS Avançado** da Origamid.

A aplicação consiste em um **site institucional para uma cafeteria**, com foco em apresentar produtos, informações da marca e um layout moderno.

O diferencial deste projeto é que o mesmo layout foi estilizado utilizando **diferentes abordagens de CSS**, permitindo comparar técnicas de construção de layout na web.

A versão publicada no **GitHub Pages** foi construída utilizando **Bootstrap**, enquanto na pasta `layouts` do repositório é possível encontrar outras versões do mesmo layout desenvolvidas com diferentes técnicas de estilização.

---

### Tech Stack:

- **HTML**
- **CSS (Avançado)**
- **Bootstrap**

---

### Objetivo do Projeto:

- Praticar técnicas avançadas de **CSS**
- Entender diferentes abordagens para construção de **layouts web**
- Comparar métodos de estilização utilizando:
  - **Float**
  - **Flexbox**
  - **CSS Grid**
  - **Bootstrap**
 
---

### Aprendizados:

- Construção de layouts utilizando **diferentes técnicas de CSS**
- Comparação entre abordagens tradicionais e modernas de layout
- Utilização do **Bootstrap** para criação de interfaces responsivas
